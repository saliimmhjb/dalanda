# Dalanda 🧠

> AI-powered Human Resource Management System

Dalanda is a full-stack HR management application built with **Flutter** and **Django REST Framework**, featuring an integrated conversational AI assistant powered by **Ollama** and **Llama 3.2**.

---

## Features

- 👥 Employee Management
- 🗓️ Leave Tracking
- 📅 Meeting Scheduling
- 📊 HR Analytics
- 🔔 Notifications
- 🤖 Conversational AI Assistant (Ollama + Llama 3.2)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Flutter |
| Backend | Django REST Framework |
| AI Assistant | Ollama + Llama 3.2 |
